# Problemas con ftp

## Error de usuario o contraseña

Si el cliente pone un usuario que no existe en el servidor o pone la contraseña de este mal le saltara un error 

### Solución 

Comprobar que el usuario es correcto y asegurarse de escribir bien la contraseña
Usuario = bchecker
Password = bchecker121

## Problemas con el listado de archivo o transferencia

Si el usuario se conecta correctamente, pero no puede ver los archivos o las subidas/descargas fallan lo mas posible es que sean por tema de permisos

### Solución

Comunicar al SysAdmin del servidor que compruebe los permisos de lectura/escritura del usuario
