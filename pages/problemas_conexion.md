# Problemas con la conexión

## Error de conexión

Si al momento de intentar buscar la web donde se encuentra alojada la base de datos esta no carga lo mas posible es que sea un tema de conexión

### Solución

Comprobar la conexion con el servidor 

```bash
ping 192.168.60.1
```
Si hace conexión comprobar que se pone correctamente la URL

```bash
http://192.168.60.1

```
